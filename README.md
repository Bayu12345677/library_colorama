- ### library colorama

- [![](https://img.shields.io/github/issues/Bayu12345677/library_colorama)](https://youtube.com/channel/UCtu-GcxKL8kJBXpR1wfMgWg)


cara menggunakan
```bash
source colorama
biru=$(bold biru)

if curl -s google.com 2>/dev/null 1>/dev/null; then {
      printf "%shalo bumi" ${biru}
   }
fi
```

sesuaikan `patch` makefile di terminal anda

```makefile
PATCH = </patch/usr/bin/>
```

- `setup`
```
make install
```

- `uninstall`

```
make uninstall
```


- `*list*`
```
bold mode
normal mode
stop, break color
```
`how use`
> bold <warna> =! bold biru

- example
```bash
#!/bin/bash

source colorama

bold=$(bold merah)
normal=$(normal biru)
stop=$(stop)

printf "%sHalo manusia %sbumi${stop}" ${bold} ${normal}

: '
color list :
|_____________________|
|        hitam        |
|        merah        |
|        hijau        |
|        biru         |
|        kuning       |
|        pink         |
|        cyan         |
|        putih        |
=======================
'
```
